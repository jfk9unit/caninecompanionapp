# CanineCompass Social Media Setup Guide

## Why Create Social Media Accounts?

Creating official social media accounts for CanineCompass will:
- Allow users to share training progress directly to your profiles
- Build community and engagement
- Increase app visibility and downloads
- Enable direct communication with users

---

## Accounts to Create

**URL:** https://www.facebook.com/pages/create

**Steps:**
1. Go to Facebook and click "Create Page"
2. Choose "Business or Brand"
3. Page Name: **CanineCompass**
4. Category: "Pet Service" or "App Page"
5. Add profile picture (dog icon/logo)
6. Add cover photo (app screenshot or dog image)

## After Creating Accounts

Once you've created these accounts, update the social sharing links in the app:

**File:** `/app/frontend/src/components/SocialShare.jsx`

Update these URLs:
```javascript
tiktok: {
  getShareUrl: () => `https://www.tiktok.com/@YOUR_TIKTOK_USERNAME`
},
instagram: {
  getShareUrl: () => `https://www.instagram.com/YOUR_INSTAGRAM_USERNAME/`
}
```
